# @envsetup/cli

Standardize and automate your development environments with AI-powered orchestration.

[![NPM Version](https://img.shields.io/npm/v/@envsetup/cli/alpha?style=flat-square)](https://www.npmjs.com/package/@envsetup/cli)
[![NPM Downloads](https://img.shields.io/npm/dm/@envsetup/cli?style=flat-square)](https://www.npmjs.com/package/@envsetup/cli)

## 🚀 Features

- **Instant Initialization**: Set up full-stack projects in seconds with `envsetup init`.
- **Professional Templates**: Pre-configured Docker setups for Next.js, FastAPI, Go, and Express.
- **System Doctor**: Verify your system prerequisites with `envsetup doctor`.
- **Infrastructure Orchestration**: Manage multi-container environments reliably.

## 📦 Installation

Install the CLI globally using NPM:

```bash
npm install -g @envsetup/cli@alpha
```

## 🛠️ Usage

### ⚙️ Initialize a Project
Create a new standardized environment in your current directory:

```bash
envsetup init
```

### 🩺 System Check
Verify if Docker, Node.js, and other tools are correctly installed:

```bash
envsetup doctor
```

## 🏗️ Supported Stacks

- **Next.js** (Node.js/Alpine)
- **FastAPI** (Python/Slim)
- **Go** (Alpine)
- **Express** (Node.js/Alpine)
